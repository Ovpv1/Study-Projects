# Delivery Analyzer
This software was made to simulate a list of deliveries. It receives the user's input and then calculates the price of each delivery (also calculating its total value).
## Alert ⚠️
This may not be the latest version. Check for new updates [here](https://github.com/Ovpv1/DeliveryAnalyzer)

## Skills Used / Learned
- Algorithm Development: Creating complex logic to calculate tiered pricing and conditional bonuses.
- Input Validation: Implementing robust 'while' loops and error handling to ensure user inputs are valid and the system doesn't crash.
- Data Structures: Managing and organizing information using Arrays and Objects to represent delivery records.
- Problem Solving: Identifying potential business needs and transforming the solution into functional code.
- Clean Code Practices: Organizing the project into small, reusable functions with clear naming conventions for better readability.
