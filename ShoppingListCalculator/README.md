# Shopping List Discount Checker
This program simulates a system for a fictional supermarket to calculate the total price of the shopping list and, if it is in the requirements, apply a 15% discount on the total value
## How to run it?
Since I don't know yet how to run .js files directly in the computer, I made it to run in an online compiler
## Skills used/learned:
- How to iterate and verify the values in a array
- How to implement functions
- Change different variable types
- Preventing bugs from invalid inputs
